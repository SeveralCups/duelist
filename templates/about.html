About This Site.

A tale of incomprehensible excitement...

I built this site as one of my final projects for The App Brewery's python bootcamp course on Udemy. I decided to force as many terrible list puns as possible while making the site, which turned out to be just two... yeah. Duelist is a horrible and extremely forced name, which is why it's so perfect. Using "enlist" as much as possible was a genuine stroke of genius though, don't @ me ( or do ).

The site is built using Flask, Bootstrap 5, WTForms, SQLAlchemy and a PostgreSQL database. If you make an account, your password is salted and hashed and all that good stuff, so don't worry. The guide ( which is actually rendered from a list in the database ) does a good job of covering all site functionality if you would prefer not to make an account, though.

Thanks for checking out my project. Please hire me and whatnot.
